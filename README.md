# WebSocket Protocol
- WebSockets allow for full-duplex communcation
- Websocket is a seprate protocol from http
- Persistent connection between client and server

   () Client -> Server (My new message)
   () Server -> Client (Other users message)
   () Server -> Client (Other users message)
   () Server -> Client (Other users message)
